How am I loading new images on every refresh?




I am using the unsplash developers API
Everytime the page makes a new request, unsplash will send a new image according to the query
The request is made by the following link using it as the value for "src" attribute in the "img" tag
https://source.unsplash.com/{width}x{height}/?{search keyword}"
